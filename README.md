## Usage

1. Clone the repo
   ```sh
   git clone https://github.com/DogDogHot/make-nft-json.git
   ```
2. custom json data
   ```sh
   cd make-nft-json && vi index.js
   ```
   ```js
   const name = "custom";
   const description = "custom";
   const imageURL = "customURL";
   const total = Number;
   ```
3. generate json
   ```sh
   node index.js
   ```
